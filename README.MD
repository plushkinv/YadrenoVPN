## 1. 🚀 Регистрация и создание VPS-сервера

### Выбор хостинга

Для работы VPN-сервера нам нужен VPS (виртуальный сервер) с **Ubuntu 22**. 

**Рекомендуемый хостинг:** [VDSina](https://www.vdsina.com/?partner=7wex54cw4j5u) — надёжный провайдер с дата-центрами в России и Европе.

> 💡 **Бонус:** При регистрации по [этой ссылке](https://www.vdsina.com/?partner=7wex54cw4j5u) вы получаете **скидку 10%** на все услуги!

### Шаг 1: Регистрация на VDSina

1. Перейдите на [VDSina](https://www.vdsina.com/?partner=7wex54cw4j5u)
2. Нажмите кнопку **"Регистрация"** в правом верхнем углу
3. Заполните форму регистрации (email, пароль)
4. Подтвердите email, перейдя по ссылке в письме
5. Пополните баланс любым удобным способом

### Шаг 2: Создание VPS-сервера

1. В личном кабинете нажмите **"Создать сервер"**
2. Выберите **локацию** — для VPN лучше выбрать **зарубежный дата-центр** (Амстердам, Франкфурт)
3. Выберите **операционную систему: Ubuntu 22.04 LTS**
4. Выберите **тариф** — для начала достаточно минимального:
   - **1 vCPU**
   - **1 ГБ RAM** 
   - **10 ГБ SSD**
   
   > Этого хватит для ~20-50 одновременных пользователей VPN

5. Нажмите **"Создать"** и дождитесь запуска сервера (обычно 1-2 минуты)
6. После создания вы получите:
   - **IP-адрес** сервера
   - **Логин** (обычно `root`)
   - **Пароль** для SSH-подключения

> ⚠️ **Важно:** Сохраните эти данные в надёжном месте! Они понадобятся для настройки VPN-панели.

### Шаг 3: Подключение к серверу

Для подключения к серверу используйте SSH-клиент:

**Windows:** 
- Встроенный терминал (PowerShell): `ssh root@ваш_ip`
- Или программа [MobaXterm](https://mobaxterm.mobatek.net/)

**Linux/macOS:**
- Терминал: `ssh root@ваш_ip`

При первом подключении:
1. Подтвердите добавление ключа сервера (введите `yes`)
2. Введите пароль из письма от хостинга

#### 📝 Редактирование текстов
В админ-панели можно менять тексты и ссылки без перезагрузки бота:
1. **Текст главной страницы**: Приветствие при команде `/start`.
2. **Текст справки**: Содержимое раздела `/help`.
3. **Ссылки**: Настройка URL для кнопок «Новости» и «Поддержка».

##### Специальные теги для текста главной страницы

В тексте главной страницы можно использовать теги для управления выводом списка тарифов:

| Тег | Поведение |
|-----|-----------|
| *(тег не указан)* | Список активных тарифов **автоматически добавляется в конец** сообщения |
| `%тарифы%` | Список тарифов вставляется **в указанное место** текста |
| `%без_тарифов%` | Тарифы **не выводятся** совсем (тег удаляется из текста) |

> 💡 **Пример:** Чтобы разместить тарифы между двумя абзацами, вставьте `%тарифы%` в нужное место. Если тарифы вообще не нужны на главной — добавьте `%без_тарифов%` в любом месте текста.

## 3. 🛠️ Установка и запуск (Linux/Ubuntu)

### Шаг 1: Подготовка системы
Обновите пакеты и установите необходимые зависимости:
```bash
sudo apt update && sudo apt install -y python3-pip git
```

### Шаг 2: Клонирование репозитория
Скачайте проект с GitHub:
```bash
git clone https://github.com/plushkinv/YadrenoVPN.git
cd YadrenoVPN
```

### Шаг 3: Установка зависимостей Python
Используйте `pip` для установки библиотек:
```bash
pip install -r requirements.txt
```

### Шаг 4: Настройка
1. Скопируйте файл шаблона настроек:
   ```bash
   cp config.py.example config.py
   ```
2. Отредактируйте `config.py`, указав ваши данные:
   - `BOT_TOKEN`: Токен вашего бота от @BotFather.
   - `ADMIN_IDS`: Ваш Telegram ID.

> [!IMPORTANT]
> Файл `config.py` содержит секретные данные, поэтому он автоматически игнорируется Git и не попадет в публичный репозиторий. Всегда используйте `config.py.example` как образец для новых установок.

### Шаг 5: Запуск
#### Способ 1: Обычный запуск (для тестов)
Запустите бота через Python:
```bash
python3 main.py
```

> ℹ️ **Примечание:** При старте бот проверит версию базы данных и выведет в лог информацию о статусе миграций (успех/ошибка).

#### Способ 2: Автозапуск через systemd (рекомендуется)
Чтобы бот работал постоянно и запускался сам после перезагрузки сервера:

1. Копируем файл сервиса:
   ```bash
   cp yadreno-vpn.service /etc/systemd/system/
   ```
2. (Опционально) Если вы установили бота не в `/root/YadrenoVPN`, отредактируйте путь в файле:
   ```bash
   nano /etc/systemd/system/yadreno-vpn.service
   ```
3. Активируем и запускаем:
   ```bash
   systemctl daemon-reload
   systemctl enable yadreno-vpn
   systemctl start yadreno-vpn
   ```
